

## Getting started
```bash
$ git clone https://github.com/Techies-Rw/issue-backend 
```
```bash
$ npm install
```

## Running the app


#### development
```bash
$ npm run dev
```
#### watch mode

```bash
$ npm run dev
```
#### production mode

```bash
$ npm run start:prod
```

## Test

#### unit tests
```bash
$ npm run test
```

#### e2e tests
```bash
$ npm run test:e2e
```

#### test coverage
```bash
$ npm run test:cov
```
